E-Commerce Flower Shop Website

Built using HTML & CSS

A beautifully designed Flower Shop Website that showcases products, categories, and a modern shopping layout.
This project focuses on building a clean UI using only HTML and CSS, ideal for beginners learning web design and layout techniques.
Features:

 Modern and Attractive UI

 Responsive Design (mobile-friendly)

 Homepage with banners & featured flowers

 Product listing section

 Product cards with price & buttons

 Navbar + Footer + Contact Section

 Pure HTML + CSS (no JavaScript)

 Technologies Used:

HTML5 – Structure of the website

CSS3 – Styling, layout, responsiveness
